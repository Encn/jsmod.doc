jsmod/ui/pagination

new require("jsmod/ui/pagination")(element, option)

分页控件，无需写 html ，提供一个 div 节点自动生成所有的分页所需标签
Parameters:
Name Type Description
element dom | string 分页控件的容器
option object 分页控件配置参数
Properties
Name Type Argument Default Description
pageCount int 一共有多少页
currentPage int <optional>
 0 当前页
maxShowPage int <optional>
 10 最多显示分页个数
textLabel array <optional>
 new Array('首页', '上一页', '下一页', '尾页') 几个特殊关键字
pageLabel string | function <optional>
 {#0} 字符串用 {#0} 代表当前页, 函数则取返回值作为显示。函数其参数 page 为索引计数（起始0）；而替换字符串为 page + 1
preventInitEvent bool <optional>
 false 是否阻止初始化时触发事件
