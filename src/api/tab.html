jsmod/ui/tab

new require("jsmod/ui/tab")(element, option)

初始化时需要有特定的HTML结构，被标识为 mod-tab-item-active 的为当前开启状态，默认为第一个。 当存在 data-target 属性时，会将当前非开启状态的 target 隐藏（当然也可以手动执行）
Parameters:
Name Type Argument Description
element dom | selector 需要实现 tab 的 dom 元素
option object <optional>
 可配置项
Properties
Name Type Argument Default Description
className object <optional>
 自定义 className
trigger string <optional>
 click 触发切换 tab 的事件，可选项：click, hover
delay Delay <optional>
 200 当选择 hover 可用, 激活某个 tab 的延迟
preventInitEvent bool <optional>
 false 是否阻止初始化时触发事件
