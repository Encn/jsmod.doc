Module: jsmod/ui/fixElement

固定位置元素模块

new (require("jsmod/ui/fixElement"))(element, option)

绝对定位元素为 jsmod 大多数 ui 控件基类；同时也可被外部直接使用或继承使用。 其提供了将 dom 元素（html片段）定位到页面特定位置（坐标，元素附近）的基础支持
Parameters:
Name Type Description
element string | dom | selector 需要定位的元素，可以是 html, dom, 选择器
option object 配置参数
Properties
Name Type Attributes Default Description
target dom | selector <optional>
 定位到这个元素附近 (取选择器中获取的第一个元素作为定位元素)
targetType string <optional>
 用 "," 分割可传入一至三个值，每个值都可选 top, right, bottom, left, center
offset Coords <optional>
 定位时的偏移
position Coords <optional>
 没有 target 参数时, 采取绝对定位方式时传入的坐标
fixed bool <optional>
 false 是否使用 fixed 定位, 仅当采用坐标定位时可用, ie6 会自动加入 hack 支持 fix 定位
zIndex int <optional>
 1000 元素的 z-index
preventShow bool <optional>
 false 是否阻止初始化时显示元素
preventResize bool <optional>
 false 是否阻止resize时重定位元素
appendInBody bool <optional>
 false 如果 target 的父元素设置 overflow:hidden 且 relative, absolute 定位时，元素会被隐藏，设置此值为 true 元素将成为 body 的子元素

Example
var FixElement = require("jsmod/ui/fixElement"), fixEl;

// 创建实例 (绝对定位)
new FixElement('<div class="module">Hello World!</div>', {
    position: {left: 100, top: 200}
});

// 创建实例 (元素附近)
new FixElement('<div class="module">Hello World!</div>', {
    target: "#create-target"
});
