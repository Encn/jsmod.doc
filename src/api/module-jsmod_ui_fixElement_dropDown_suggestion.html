Module: jsmod/ui/fixElement/dropDown/suggestion

suggestion 提供ajax调用远程数据返回结果，并自动调用下拉框

new (require("jsmod/ui/fixElement/dropDown/suggestion"))(option)

使用时最好使用json请求，jsonp需要保证支持标准回调函数，不然需要在实例中重写getData方法

Properties
Name Type Attributes Default Description
url string | function 请求数据的地址。如果是函数时会将target中输入作为参数传递；会用返回的数据作为ajax的地址
target dom | selector 定位到这个元素附近，此target必须为可输入的input
fun function 会将index, item作为参数传递。返回数据结构为 DropItem
sendData function 会将target中输入作为参数传递。会用返回的数据作为ajax的数据
handleData function 会将异步返回数据作为参数传递。需要返回数组作为渲染数据
jsonpOption JSONPOption 如果请求是jsonp时的配置

 当没有输入时下拉框显示的内容

preventShow bool <optional>
