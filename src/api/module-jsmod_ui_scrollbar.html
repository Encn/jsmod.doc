(require("jsmod/ui/scrollbar"))(element, option)

滚动条组件，无需改变容器的形态即可实现自定义滚动条 解决了各个浏览器滚动条不同宽度的困扰，absolute 定位，无需改变盒模型 被传入的容器必须有且只有一个子元素，不然会报错 且该子元素的定位属性会被设置为 relative
Parameters:
Name Type Description
element string | dom 需要创建滚动条的容器
option object 配置选项
Properties
Name Type Attributes Default Description
offsetRight int <optional>
 1 滚动条距离右侧的位置
interval int <optional>
 50 每次滚动的偏移
autoHide bool <optional>
 false 是否自动隐藏滚动条
className string <optional>
 自定义滚动条容器类名
scrollbarNodeStyle object <optional>
 添加至滚动按钮的样式
scrollbarNodeHoverStyle object <optional>
 添加至滚动按钮滚动中、hover时的样式
Source:
  • scrollbar.js, line 247
Example
// 默认滚动样式
scrollbarNodeStyle: {
    "background": "#666",
    "border-radius": "3px",
    "opacity": "0.8",
    "width": 3,
    "border": "1px solid #444"
}

// 默认滚动中 / hover 样式
scrollbarNodeHoverStyle: {
    "background": "#333"
}
